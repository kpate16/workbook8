# REST APIs

## Starting

To start an API run one of the following commands.

```
npm run placeholder
```

## Stopping

### Windows

With your cursor in the terminal.

```
Ctrl+C
```

### MacOS

With your cursor in the terminal.

```
Cmd+C
```
